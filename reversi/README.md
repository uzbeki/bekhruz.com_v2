## üßê About <a name = "about"></a>

Reversi is a strategy board game for two players, played on an 8√ó8 uncheckered board. It was invented in 1883. Othello, a variant with a change to the board's initial setup, was patented in 1971.

## üèÅ Getting Started <a name = "getting_started"></a>

If you just want to play the game with a friend, just click on [the demo page](https://uzbeki.github.io/Reversi/).
